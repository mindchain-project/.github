# MINDCHAIN PROJECT

Mindchain est une DApp qui a pour but de permettre à tout créateur d’images assisté par l’IA de :

- Créer, itérer et finaliser ses œuvres générées par IA ;
- Authentifier sa création via un smart contract ERC-721 ;
- Émettre un certificat NFT $ORIGYN attestant de l’authenticité et de la traçabilité du processus créatif.

MindChain agit comme une interface sécurisée entre les modèles d’IA générative, la blockchain et le wallet Web3 de l’utilisateur.
